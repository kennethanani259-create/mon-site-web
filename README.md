# 💖 SkinGlow Beauty ✨

**App web féminine rose/mauve pour analyse teint & recommandations beauty luxe!**

## 🎀 Fonctionnalités Magiques
- 📸 **Analyse Teint**: Upload selfie → IA Canvas détecte teint/sous-ton/type/problèmes
- 💕 **Profil Glow**: Objectifs personnalisés, historique magique localStorage
- 🌟 **Recos L luxe**: Glossier, Dior, Fenty, Rare Beauty - filtrables prix/marque
- 🎨 **Design Artistique**: Rose/mauve gradients animés, fonts cursive, responsive mobile-first

## 🚀 Quick Start
```
start SkinGlowBeauty/index.html
```
1. Upload photo visage 
2. Clique zone joues (ROI rose)
3. Analyse → Glow results 💖
4. Choisis objectif → Recos personnalisées ✨

## 🛠 Tech Pure Web
- **HTML5/CSS3/JS** Canvas pour "IA" teint (HSV + variance)
- **Responsive** Mobile-first, touch-optimized
- **Data** JSON produits luxe féminins
